Proyectos

Proyecto - Web App de Microservicios en AWS para Clima. Tráfico y Noticias

Desarrollé una aplicación web de microservicios centrada en AWS que permite a los usuarios consultar el pronóstico del clima en cualquier país, visualizar el tráfico en tiempo real mediante un mapa interactivo y acceder a noticias globales. La aplicación fue implementada utilizando diversos servicios de AWS, incluyendo EC2, RDS, Amplify, VPS y S3. Además, configuré pipelines de CI/CD para realizar actualizaciones en tiempo real. El backend fue desarrollado en React, el frontend en Python, y la base de datos se alojó en Docker. Este proyecto me brindó experiencia profunda en despliegue en la nube, DevOps y arquitectura de microservicios con AWS como plataforma principal.

Proyecto - Migración de Datos y Web App para Gestión Bancaria

En este proyecto, migré una base de datos bancaria desde múltiples archivos Excel a PostgreSQL y luego la implementé en Docker. Además, desarrollé una aplicación web con vistas separadas para clientes y administradores. Los clientes podían gestionar sus solicitudes de préstamos, mientras que los administradores validaban pagos y revisaban solicitudes. El backend fue construido en React y el frontend en Python, brindando una experiencia eficiente y fluida. Esta experiencia fortaleció mis habilidades en desarrollo full stack y administración de bases de datos.
